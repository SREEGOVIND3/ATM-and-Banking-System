# SRS (Procedure):

## Step 1: Problem Identification

The Bank and ATM System is used to provide secure and quick banking services like deposit, withdrawal, balance checking, and loan processing.

## Step 2: Requirement Analysis
Functional Requirements:
Login and create account,
Validate card and PIN,
Deposit and withdraw cash,
Check balance and print receipt,
Loan request and approval.

Non-Functional Requirements:
Security,
Reliability,
Fast performance,
Easy to use.

## Step 3: System Design

Draw UML diagrams to represent the system:      
Use Case Diagram      
Class Diagram      
Activity Diagram      
Sequence Diagram      
Communication Diagram      
Package Diagram      
Design the database with tables for Customer, Account, and Transaction.
